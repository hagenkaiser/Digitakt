# Tonverk Panel Layout Reference (Pages 14-15)

## 3. PANEL LAYOUT

### 3.1 FRONT PANEL

The front panel diagram shows numbered elements 1-29 with the following controls:

**Top Row (Left to Right):**
- Jacks: A out, B out, 1-Out-5, A-In-B, MIDI In, MIDI Out, MIDI Thru, USB 1-2, Power

**Main Controls:**

1. **MAIN VOLUME** - Sets the volume for the A/B outputs and the Headphones outputs.

2. **[PRESET]** - Opens the LOAD PRESET section of the FILE menu. Secondary function opens the FILE menu.

3. **[ARPEGGIATOR]** - Opens the ARPEGGIATOR menu. Secondary function toggles the arpeggiator on/off.

4. **[SETTINGS]** - Opens the SETTINGS menu (project management, MIDI configuration, system settings). Secondary function saves the active project.

5. **[CHORD]** - Opens the CHORD/SCALE SETUP menu. Secondary function toggles CHORD mode on/off.

6. **[SAMPLING]** - Opens the SAMPLING menu (recording audio tools). Secondary function opens the Sample browser.

7. **[SONG]** - Opens the Song selection screen. Secondary function toggles SONG mode on/off.

8. **[TEMPO]** - Opens the TEMPO menu (tempo, swing, metronome). Secondary function enables tapping the tempo.

9. **[MUTE]** - Accesses the track mute functionality. Secondary function accesses the ROUTING menu.

10. **Screen** - Main display

11. **[STEP EDIT]** - Accesses the Step Edit functionality.

12. **[+]/[-]** - Transposing the KEYBOARD. <KEYBOARD OCTAVE> LEDs show currently selected octave. Secondary functions: sequencer track transpose and TRANSFORM menu.

13. **DATA ENTRY knobs A-H** - Used for setting parameter values. Press and turn to change values in larger increments.

14. **[PARAMETER PAGE] keys** - (TRIG, SRC, FLTR, AMP, FX, MOD) Access the PARAMETER pages of the active track. Secondary functions access different sections of the TRACK SETUP menu.
    - **[TRIG PARAMETERS]** - Parameters: NOTE, VELOCITY, other trig-related. Secondary: track and keyboard settings in TRACK SETUP.
    - **[SRC]** - SRC pages with CHANNEL, PROGRAM, AFTERTOUCH (for MIDI tracks). Secondary: SRC machine selection in TRACK SETUP.
    - **[FLTR]** - FILTER pages (Base-width, Multimode filters). On MIDI: CC value and first 8 assignable CC commands. Secondary: filter settings in TRACK SETUP.
    - **[AMP]** - AMP page (amplitude envelope). On MIDI: CC value and 8 additional assignable CC commands. Secondary: amp settings in TRACK SETUP.
    - **[FX]** - FX pages (FX machines parameters, effect send levels, track routing). Secondary: FX machine selection in TRACK SETUP.
    - **[MOD]** - LFO and Mod Envelope parameters for the active track.

15. **[YES]** - Entering sub-menus, selecting, and confirming.

16. **[FILL]** - Activates FILL mode. Secondary function accesses the FILL EDIT menu.

17. **[ARROW] keys** - [UP], [DOWN], [LEFT], [RIGHT] for navigation and parameter values.
    - [UP] secondary: SEQUENCE LANE menu
    - [DOWN] secondary: TRIG MODE menu

18. **[PAGE]** - Selects the active pattern page (if >16 steps). Secondary function accesses PAGE SETUP menu.

19. **[NO]** - Exiting an active menu, backing up one step, and negating.

20. **[TRIG] keys** - Enter/remove sequencer trigs and parameter locks (with DATA ENTRY knobs). Also select track, bank, pattern, and song (with [TRK], [PTN], [SONG] keys).
    - Lit red keys = trigs on sequencer
    - Flashing red/yellow keys = parameter locks
    - In GRID RECORDING or LIVE RECORDING: light moves along 16 steps (up to 16 pages)

21. **[KEYBOARD] (bottom row) / [SUBTRACKS]** - Select subtracks when Subtracks machine is selected. Secondary: select samples for subtracks. Together with top row [KEYBOARD]: play presets and assign note values. Called **[KEYBOARD C1-C2]** from left to right.

22. **[KEYBOARD] (top row)** - Together with bottom row: play presets and assign note values. Called **[KEYBOARD C1#-A1#]**.
    - **[KEYBOARD D#1]** secondary: recall previously memorized state
    - **[KEYBOARD D#1]** memorizes state of active pattern
    - **[KEYBOARD F#1]** toggles metronome on/off
    - **[KEYBOARD G#1]** opens SCALE menu
    - **[KEYBOARD A#1]** opens SCALE menu

23. **[STOP]** - Stops playback. Secondary function is paste operation.

24. **[PLAY]** - Starts sequencer playback. Press again to pause. Secondary function is clear operation.

25. **[RECORD]** - Activates/deactivates GRID RECORDING mode. [RECORD] + [PLAY] activates LIVE RECORDING mode. Secondary function is copy operation.

26. **[FUNC]** - Press and hold, then press another key to access secondary function. Mint green text on front panel shows secondary functions.

27. **[TRK]** - Press [TRK] + [TRIG] keys to select a track for editing. Secondary function accesses PERFORM mode.

28. **[PTN]** - Selects bank and pattern with [KEYBOARD] (bottom row) and [TRIG 1-16] keys.

29. **LEVEL/DATA** - Sets overall volume level of active track. Also used for setting parameters and scrolling through lists.
